Colors

Правила использования цветов

Основные цвета сайта

  • Черный #252525. Используется для надора основного текста.
  • Белый #FFF. Основной цвет фона страниц.
  • Основной синий #5070B2. Фирменный синий цвет.
  • Основной зеленый #30C161. Используется в основном у кнопок подтверждения (акцептования) какого либо действия.
  • Оранжевый #FEC200. Основной цвет СТА кнопок.
  • Акцентный синий #4BACFF. Используется для визуальных акцентов и у акцентных кнопок (не СТА).
  • Цвет ссылок #3366CC.
  • Красный #EE4200. Для аллертов и иного привличения внимания к критически важной информации на сайте.
